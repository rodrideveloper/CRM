## Buyer Personas

### Persona 1: "Nico" — Vendedor independiente (PRIMARIO)

| Campo | Detalle |
|-------|---------|
| **Nombre ficticio** | Nicolás, 32 años |
| **Ocupación** | Vendedor de seguros / corredor inmobiliario / venta de autos usados |
| **Ubicación** | Conurbano / ciudad intermedia, Argentina |
| **Ingresos** | Variables, $500K-1.5M ARS/mes en buenos meses |
| **Dispositivo** | Android gama media, todo desde el celular |
| **Redes** | Instagram, TikTok, WhatsApp (su oficina) |
| **Día típico** | Recibe 5-10 consultas por día. Las anota mentalmente o en un chat consigo mismo. Contacta a los "calientes" y se olvida del resto. Fin de mes no sabe cuántos perdió |
| **Dolor central** | "Sé que pierdo ventas pero no tengo tiempo de organizar todo" |
| **Objeción principal** | "Ya probé apps y son todas complicadas" / "No tengo tiempo" |
| **Trigger de compra** | Perder una venta grande que tenía "en la cabeza" |
| **Canales de captación** | TikTok (contenido de dolor), Instagram (reels), boca a boca |

**Frase que diría:**
> "Tengo como 30 personas que me consultaron y no les volví a escribir. Pero bueno, algún día me organizo."

### Persona 2: "Caro" — Emprendedora por redes (PRIMARIO)

| Campo | Detalle |
|-------|---------|
| **Nombre ficticio** | Carolina, 28 años |
| **Ocupación** | Vende ropa/accesorios/productos por Instagram y WhatsApp |
| **Ubicación** | CABA / GBA, Argentina |
| **Ingresos** | $200K-600K ARS/mes |
| **Dispositivo** | iPhone o Android gama media, vive en el celular |
| **Redes** | Instagram (su vidriera), TikTok, WhatsApp (cierra ventas ahí) |
| **Día típico** | Publica stories, responde DMs, pasa gente a WhatsApp. Tiene 200+ chats abiertos. No sabe quién le preguntó por qué producto. No hace seguimiento |
| **Dolor central** | "Me escriben muchos pero después no sé quién es quién" |
| **Objeción principal** | "No necesito algo tan serio, yo vendo por redes nomás" |
| **Trigger de compra** | Ver que una colega emprendedora lo usa y le va mejor |
| **Canales de captación** | TikTok (humor/identificación), Instagram (reels), comunidades emprendedoras |

**Frase que diría:**
> "Ayer una clienta me dijo 'ya le compré a otra porque nunca me respondiste'. Me quería morir."

> "Le pago el sueldo a 3 vendedores y no tengo idea si están haciendo seguimiento o mirando el techo."

## Targeting para Meta Ads

### Audiencia Fase 1 (Nico + Caro)

```
Edad: 22-45
Género: Todos
Ubicación: Argentina (foco Buenos Aires, Córdoba, Rosario, Mendoza)
Idioma: Español

Intereses (OR):
- Emprendimiento
- Ventas
- Marketing digital
- Negocios pequeños
- E-commerce
- Instagram for Business
- Marketplace (Facebook)
- Venta online
- Dropshipping
- Seguros (para vendedores)
- Real estate / inmobiliaria (para corredores)

Comportamientos:
- Propietarios de pequeñas empresas (Meta lo detecta)
- Usuarios de dispositivos móviles (todos)

Exclusiones:
- Empleados de empresas +500 personas
- Intereses en Enterprise software / Salesforce / HubSpot
```

## Criterio de decisión: ¿Una feature nueva le sirve a nuestro usuario?

Ante cualquier feature nueva, preguntarse:

1. ¿"Nico" la usaría desde el celular en 30 segundos?
2. ¿"Caro" la entendería sin tutorial?
3. ¿Resuelve el problema de **perder ventas por falta de seguimiento**?
4. Si la respuesta a cualquiera es NO → la feature puede esperar.
